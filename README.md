# Flask Kantine

Min Flask nettside for kantinen som vi ikke har. Den har ukens meny, varer og kontaktinformasjon.

## Hva den gjør

- Viser dagens rett automatisk basert på hvilken dag det er
- Har en meny med alle dagene i uken
- Viser varer med priser
- Kontaktside med ansatte

## Hvordan kjøre den

1. Installer Flask: `pip install flask`
2. Kjør: `python app.py`
3. Gå til `http://localhost:5000`

## Filer

- `app.py` - Hovedfilen med alle rutene
- `templates/` - HTML-sidene
- `static/css/style.css` - Stylingen
- `static/images/` - Bildene

Debug-modus er på så du slipper å restarte serveren hver gang du endrer noe.
